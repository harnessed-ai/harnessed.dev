Two tmux plugins

Panes that remember. Panes that get your attention.

Two small tmux plugins for engineers running AI coding agents in panes. tmux-ai-sessions-restore brings your Claude Code and Kiro CLI conversations back after a reboot — not just the layout. tmux-nudge blinks the one pane that needs a human next, whether that's an agent, a build, or a script.

tmux-ai-sessions-restore

Resume the conversation, not just the layout

Makes claude and kiro-cli chat relaunch resumed, in the same pane, where you left off — after tmux-resurrect brings the pane back.

Learn more →

tmux-nudge

The pane that needs you, blinking

Blinks a label on the pane and its status-bar entry the moment it needs a human — an AI agent, a finished build, or anything else.

Learn more →
Install

Both plugins install through TPM

tmux-ai-sessions-restore

set -g @plugin 'tmux-plugins/tmux-resurrect'
set -g @plugin 'bmohan01/tmux-ai-sessions-restore'
set -g @plugin 'tmux-plugins/tmux-continuum'
run '~/.tmux/plugins/tpm/tpm'
